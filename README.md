# Proyecto Web: Asambleas de Dios - Guinea Ecuatorial

Este repositorio contiene la estructura de la página web oficial y la interfaz de usuario para el futuro sistema de gestión de la Organización Asambleas de Dios de Guinea Ecuatorial.

## Fases del Proyecto

1. **Fase 1: Interfaz Visual (Frontend)** - *En curso*
   - Rediseño de la página de inicio.
   - Creación de páginas institucionales (Organización, Presbiterio).
   - Creación de páginas interactivas y multimedia (Noticias, Actividades, Iglesias).
   - Creación de formularios de interacción (Ayudas, Contactos).

2. **Fase 2: Sistema de Gestión y Base de Datos (Backend)** - *Próximamente*
   - Integración de base de datos para manejar iglesias, pastores, noticias, etc.
   - Procesamiento de los formularios de contacto y solicitud de ayuda.

## Estructura del Frontend

- `index.html`: Página principal con resumen general y buscador de iglesias.
- `organizacion.html`: Página gigante detallando historia, visión, valores y estructura.
- `presbiterio.html`: Directorio y perfiles del Presbiterio Ejecutivo.
- `iglesias.html`: Directorio completo de las iglesias locales.
- `noticias-eventos.html`: Portal de noticias con integración de videos tipo podcast.
- `actividades.html`: Galería multimedia (imágenes y videos) de actividades.
- `necesidades-ayudas.html`: Sección informativa y formularios de solicitud de ayuda/donaciones.
- `contactos.html`: Información de contacto expandida, ubicaciones y redes.
- `CSS/style.css`: Hoja de estilos con variables y diseño moderno.
- `js/script.js`: Interactividad (modales, validaciones, buscadores locales).

## Tecnologías Utilizadas (Fase 1)
- HTML5 Semántico
- CSS3 (Variables, Flexbox, Grid, Animaciones)
- Vanilla JavaScript (Intersection Observer, manipulación del DOM)
- FontAwesome (Iconos)
- Google Fonts (Inter, Playfair Display)
